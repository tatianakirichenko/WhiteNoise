🧠 Концепция
WhiteNoise Sleep Timer — это простое, но эффективное приложение для создания фонового шума, помогающего заснуть. Оно позволяет:

✅ Генерировать белый, розовый или коричневый шум — научно доказанные звуки для улучшения сна.

✅ Настраивать громкость в реальном времени.

✅ Устанавливать таймер выключения (от 5 до 120 минут) с плавным затуханием.

✅ Отображать оставшееся время в удобном формате.

✅ Сохранять последние настройки (громкость, тип шума, время) для быстрого запуска.

✅ Работать в фоне (в консольных версиях — до нажатия Ctrl+C или по таймеру).

✅ Быть минималистичным и не отвлекать от сна.

🚀 Как запустить
Каждая версия использует соответствующие аудио-библиотеки. Установите зависимости и запустите.

Python
bash
pip install numpy pyaudio
python white_noise_python.py
C++
bash
# Требуется PortAudio (sudo apt install portaudio19-dev) и компилятор с C++17
g++ -std=c++17 white_noise_cpp.cpp -o whitenoise -lportaudio -lpthread
./whitenoise
Java
bash
# JavaSound встроена в JDK
javac white_noise_java.java && java white_noise_java
C# (.NET Core)
bash
dotnet add package NAudio
dotnet run
Go
bash
# Требуется PortAudio (установите порт)
go get github.com/gordonklaus/portaudio
go run white_noise_go.go
Rust
bash
cargo add cpal rand
cargo run
JavaScript (браузер)
bash
# Просто откройте white_noise_js.html в современном браузере
🧩 Пример использования
text
🌊 WhiteNoise Sleep Timer v2.0

🔮 Расширенные функции
Плавное затухание — звук постепенно угасает перед выключением (за 1 минуту до конца).

Выбор цвета шума — белый (равномерный), розовый (более низкие частоты), коричневый (глубокий гул).

Сохраняемые настройки — автоматическая загрузка последних параметров при запуске.

Минималистичный интерфейс — никаких лишних элементов, только нужные кнопки.
